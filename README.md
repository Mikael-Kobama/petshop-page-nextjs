# Petshop Homepage — Landing Page Institucional

Landing page institucional para um petshop fictício, desenvolvida com foco em apresentar serviços, gerar confiança através de depoimentos e direcionar o visitante para contato via WhatsApp. Projeto 100% front-end, sem backend, construído com Next.js.

🔗 **Demo**: [petshop-page-nextjs.vercel.app](https://petshop-page-nextjs.vercel.app/)

## 🎯 Sobre o projeto

O objetivo foi criar uma página única (one-page) que comunique credibilidade e profissionalismo para um negócio local de serviços pet, cobrindo toda a jornada de decisão do visitante: apresentação do negócio, lista de serviços oferecidos, prova social (depoimentos de clientes) e contato direto.

## 🔋 Funcionalidades

- 👉 **Hero section** com chamada principal e CTA de contato via WhatsApp
- 👉 **Seção Sobre**, com tempo de atuação no mercado e diferenciais do negócio
- 👉 **Catálogo de serviços** (Banho & Tosa, Consulta Veterinária, Táxi Pet, Hotel para Pets), cada um com descrição, duração estimada e botão de contato próprio
- 👉 **Depoimentos de clientes**, com foto, nome e relato
- 👉 **Vitrine de marcas parceiras** (Royal Canin, Golden, Premier, Fórmula Natural, Whiskas)
- 👉 **Seção de contato** com e-mail, telefone e endereço
- 👉 Layout responsivo, otimizado para mobile e desktop

## 🛠️ Tech Stack

- **Framework**: Next.js
- **Linguagem**: JavaScript / TypeScript
- **Estilização**: CSS / Tailwind CSS
- **Deploy**: Vercel

## 📌 Escopo do projeto

Este é um projeto **front-end estático**, sem integração com backend ou banco de dados — os dados de serviços, depoimentos e contato são fixos no código. O foco aqui foi inteiramente em construção de interface, responsividade, organização de componentes e boas práticas de landing page de conversão.

## 🚀 Como executar localmente

```bash
# Clone o repositório
git clone <link-do-repositorio>
cd petshop-page-nextjs

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse `http://localhost:3000` no navegador.
